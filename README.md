# Сайт "Врачи и отзывы"

Простой статический сайт для добавления врачей и отзывов о них.

Публикация на GitHub: https://github.com/Maksim-Konuhov/vraci-doctors-reviews

## Запуск

1. Откройте файл `index.html` в браузере.
3. Добавьте врача через форму "Добавить врача". Можно загрузить фото врача.
4. Выберите врача, оставьте отзыв и при желании добавьте одно или несколько фото к отзыву.
5. В поле "Ответ врача / клиники" можно указать ответ на отзыв.
6. Чтобы отредактировать существующий отзыв и добавить к нему новые фото, используйте форму "Редактировать отзыв".
7. Чтобы добавить или заменить фото врача, нажмите кнопку "Изменить фото" на карточке врача.
8. Чтобы перенести данные с локальной версии на GitHub Pages, используйте блок "Перенос данных": экспортируйте JSON с локала и импортируйте его на сайте.
9. Используйте фильтр отзывов, чтобы показать только хорошие, средние или плохие отзывы.
10. При клике или вводе в поле поиска откроется список врачей с фото, специальностью и рейтингом.

## Что делает сайт

- Сохраняет данные в `localStorage` в браузере.
- Показывает список врачей, средний рейтинг и отзывы.
- Позволяет удалять врача и его отзывы.
